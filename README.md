# compounding-learning-os

Iterative task workflow scaffold for compound-learning agents.

## Status

R&D / Experimental. Maintained by Sage / 0SxD as part of an ongoing prompt-engineering and agent-skills research portfolio.

## What this is

A portable operating system for compounding task workflows. The `dispatch_v2/` bundle captures a working loop where an agent learns from each task cycle and incrementally improves its own pre-submit gate and skills tree. Domain-agnostic core; the invariant architecture (XML directive tags, mutable state, gate evolution rule) stays constant while the domain rules get replaced for each new workstream.

- `dispatch_v2/` -- primary bundle: install protocol, playbook, skills, exemplars, adapt guide, and blank templates
